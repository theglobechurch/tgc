# The Globe Church

[![Build Status](https://travis-ci.org/theglobechurch/tgc.svg?branch=master)](https://travis-ci.org/theglobechurch/tgc)

## Setup

- `git clone`
- `npm install && npm run dev`
- `vagrant up`
- `vagrant ssh`
- `cd /srv/tgc`
- `bundle exec rails db:migrate && bundle exec rails db:seed`
- `./run-dev`
